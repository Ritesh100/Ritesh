Verify the basic operation of logic gates

NOT GATES

Introduction
NOT gate is a logic gate which implements logical negation. It have only one input and it provide the complemented input so it is also called invertor.
Functional Expression
 If X is the input in the NOT gates then the ouput of the NOT gates is Y which is X’.
e.g. Z = Z’
A = A’

Procedure
The procedure to verify the basic operation of logic OR gates are:
a. Take all apparatus
b. Give power supply to the breadboard,
c. Connect the jumper wire and give the positive and negative path to the breadboard,
d. Give number 7 a ground and 14 a high of the integrated circuit (IC) and the ground is connected to the negative whereas the high is connected to the positive,
e.Take 1 number as input and 2 as the output of an IC,
f. Take the register one leg at the output path and other leg on any point of the breadboard other than the path of the IC,
g. Put the positive leg of the led at the path of output leg of the register and negative leg at the negative path of the breadboard.

Result

After doing the same thing which is listed in the procedure, the led will glow. If the input is connected to the positive path then the led will be off and if the input is connected to the negative path then the led will glow.
